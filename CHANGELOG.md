## [0.1.0] - 2026-07-11

### Added

- Local JSONL and Hugging Face dataset loading with caching.
- Structural validation for IDs, labels, duplicate text, splits, and pairs.
- Offline JSONL, Python callable, and HTTP endpoint predictors.
- TP, FP, TN, FN, rate, ROC AUC, latency, slice, and paired-boundary metrics.
- JSON, Markdown, and privacy-minimized prediction reports.
- Promptfoo test and provider export.
- CLI, unit tests, type checks, linting, packaging, and CI configuration.
